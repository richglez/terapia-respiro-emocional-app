# Terapia Respiro Emocional CRUD APP
This is my firts Full Stack CRUD project using Angular for the frontend, Node.js and Express for the backend, and MySQL as the relational database. Implements CRUD operations through a REST API, following best development practices.

This app can do:

- CRUD Operations: create/read/update/delete Notes
- Allows a user to do login and save his personal notes
